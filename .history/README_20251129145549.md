## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- TMDB API account
- Appwrite project setup

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shalomspage/movie-discovery-app.git
   cd movie-discovery-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   
   Create a `.env` file in the root directory:
   ```env
   VITE_TMDB_API_KEY=your_tmdb_api_key_here
   VITE_APPWRITE_PROJECT_ID=your_appwrite_project_id
   VITE_APPWRITE_DATABASE_ID=your_appwrite_database_id
   VITE_APPWRITE_COLLECTION_ID=your_appwrite_collection_id
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```

## ğŸ”‘ API Keys Setup

### TMDB API
1. Visit [The Movie Database](https://www.themoviedb.org/)
2. Create an account and navigate to Settings > API
3. Generate an API key
4. Add to your `.env` as `VITE_TMDB_API_KEY`

### Appwrite Setup
1. Create an [Appwrite](https://appwrite.io/) account
2. Create a new project
3. Set up a database and collection for trending movies
4. Copy your project credentials
5. Add all credentials to your `.env` file
